# xcx-citySelect
基于腾讯地图api，读取省市区的数据进行应用~

腾讯的[WebService API][1]，在项目的要用到定位，地图等等功能时会应用到，小程序官方提供的文档是没有过多的功能来满足项目需求，所以我们可以查看一下腾讯地图的api来对应开发相关的功能，功能还是蛮多的，具体自行查看=[文档][2]和配置key秘钥。
